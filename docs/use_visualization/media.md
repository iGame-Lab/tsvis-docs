# 媒体数据使用教程
## 模块简介
媒体数据模块用于显示训练数据集的三种数据，包括音频文件，图像文件以及文本文件。用户可对媒体数据文件进行查看以及用户定制功能。
## 模块功能
1.音频文件


文件信息标识：


RUN:训练模型名称


TAG:媒体数据所属类别


STEP:迭代步数


WALL_TIME:训练绝对时间


LABEL:音频信息


![func1](./images/media/func1.png){ .img-fluid }


2.图像文件

RUN:训练模型名称


TAG:媒体数据所属类别


STEP:迭代步数

![func2](./images/media/func2.png){ .img-fluid }


3.文本文件

RUN:训练模型名称


TAG:媒体数据所属类别


STEP:迭代步数

WALL_TIME:训练绝对时间


VALUE:文本具体内容


![func3](./images/media/func3.png){ .img-fluid }


4.用户定制
用户可勾选多种数据文件到用户定制页面进行分析。
